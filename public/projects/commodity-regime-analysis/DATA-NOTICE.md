# Data notice

The project includes a retained monthly return dataset and public macroeconomic
series for reproducibility. The original adjusted-close market-price downloads
were not retained, so this release is not a complete reconstruction from raw
prices.

The source list is documented in `docs/sources.md`. Users remain responsible
for complying with the terms of the original data providers. The included
files should not be treated as an independently licensed substitute for the
providers' source data.
