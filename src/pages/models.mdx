      <ModelProject
        category="Revised empirical study"
        title="Inflation Regimes and Commodity-Producer Equity Sensitivity"
        date="May–July 2026"
        description="An audited Python and Excel study of whether the sensitivity of commodity-producer equity returns to underlying commodity returns changes when inflation is elevated."
        facts={[
          { label: 'Sample', value: '89 usable monthly observations' },
          {
            label: 'Method',
            value: 'Interaction OLS · Newey–West/HAC inference',
          },
          { label: 'Tools', value: 'Python · Excel · statsmodels' },
        ]}
        image="/models/commodity-equity-sensitivity.png"
        imageAlt="Estimated Brent, gold and copper return sensitivities in lower- and higher-inflation regimes"
        actions={[
          {
            label: 'Read methodology',
            href: '/models/commodity-regime-analysis/',
          },
          {
            label: 'Browse project files',
            href: '/projects/commodity-regime-analysis/',
          },
          {
            label: 'Download Excel model',
            href: '/downloads/commodity-regime-model.xlsx',
            download: true,
          },
          {
            label: 'Download full project',
            href: '/downloads/commodity-regime-analysis.zip',
            download: true,
          },
        ]}
        note="The original specification and write-up are retained inside the full project archive. The published conclusions reflect the corrected model."
      >
        <p>
          A data audit identified 14 observations created by forward-filling
          missing futures prices before calculating returns. The revised study
          excludes those artificial zero-return months, classifies the regime
          using the previous month's headline CPI rate and estimates
          full-sample interaction models with Newey–West standard errors.
        </p>

        <ul>
          <li>
            Brent's estimated sensitivity to energy-equity returns increased
            from −0.030 in lower-inflation months to 0.493 in high-inflation
            months; the difference was statistically significant (HAC p =
            0.016).
          </li>
          <li>
            The original claim of a reliable high-inflation increase in gold
            sensitivity was withdrawn after the correction (interaction p =
            0.680).
          </li>
          <li>
            Copper remained positively associated with mining-equity returns,
            but the estimated regime difference was not statistically precise
            at the primary 3% threshold (interaction p = 0.355).
          </li>
        </ul>
      </ModelProject>
